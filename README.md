# Teneo Protocol Node — Beta Release

**Teneo Protocol Beacon** is a cross-platform app that lets you contribute to Teneo Protocol's full stack agentic infrastructure by sharing unused internet bandwidth, while earning **Teneo Points** and **Fragments** in return.

You can contribute to the ecosystem and earn rewards through a simple loop:
**Connect**, **Earn**, **Boost**, **Repeat**.

Get set up in minutes, run your node in the background, and start accumulating rewards as each of your devices contributes to the full stack agentic infra network.

## How It Works

Install → Connect → Start earning immediately.

| Mechanic | Description |
|----------|-------------|
| **8-Hour Boost Cycle** | Claim accumulated Fragments every 8 hours to activate a reward boost |
| **Beacon Power Multiplier** | Each successful claim increases your multiplier — consistency is rewarded |
| **Decay System** | Miss a claim window and your multiplier gradually decreases |
| **Early Claim Option** | Spend Fragments to skip cooldowns and accelerate growth |
| **Multi-Device Fleet** | Every unique IP earns independently — Points AND Fragments on each device |
| **30-Day Seasons** | Compete in recurring seasons with progression resets |

**The loop: Connect → Earn → Boost → Grow**

## Install Instructions

### iOS
1. Join TestFlight: 🔜 Coming Soon (pending Apple Beta App Review)
2. Install **Teneo Protocol Node** from TestFlight
3. Log in with your Teneo account

### Android
1. Download the `.apk` file to your device
2. Open it — if prompted, enable **Install from unknown sources** for your browser
3. Tap **Install**

### Windows
1. Download the `.exe` file
2. Double-click to run
3. If you see "Windows protected your PC":
   - Click **More info**
   - Click **Run anyway**

### macOS
1. Download the `.dmg` file
2. Open the DMG and drag **Teneo Protocol Node** to Applications
3. When opening for the first time, go to **System Settings → Privacy & Security**
4. Scroll down and click **Open Anyway** next to the Teneo Protocol Node message

> The app is signed with an Apple Developer ID certificate. Notarization is being processed — once complete, this step won't be needed.

### Linux
```bash
# Ubuntu/Debian
sudo dpkg -i teneo-node_*.deb

# Fedora/RHEL
sudo rpm -i teneo-node-*.rpm
```
