# Teneo Protocol Beacon — Beta Release

**Teneo Protocol Beacon** is a cross-platform app that lets you contribute to Teneo Protocol's full stack agentic infrastructure by sharing unused internet bandwidth, while earning **Teneo Points** and **Fragments** in return.

You can contribute to the ecosystem and earn rewards through a simple loop:
**Connect**, **Earn**, **Boost**, **Repeat**.

Get set up in minutes, run your node in the background, and start accumulating rewards as each of your devices contributes to the full stack agentic infra network.

---

- ⏳ Early users have more time to accumulate rewards and multiply earnings

## How It Works

Install → Sign up → Connect → Start earning immediately.

| Mechanic | Description |
|----------|-------------|
| **8-Hour Boost Cycle** | Claim accumulated Fragments every 8 hours to activate a reward boost |
| **Beacon Power Multiplier** | Each successful claim increases your multiplier — consistency is rewarded |
| **Decay System** | Miss a claim window and your multiplier gradually decreases |
| **Early Claim Option** | Spend Fragments to skip cooldowns and accelerate growth |
| **Multi-Device Fleet** | Every unique IP earns independently — Points AND Fragments on each device |
| **Referral System** | Invite friends for 8% ongoing bonus + milestone rewards for both sides |
| **Expedition Hub** | Web dashboard at [hub.teneo.pro](https://hub.teneo.pro) — fragments, quests, referrals, fleet |
| **30-Day Seasons** | Compete in recurring seasons with progression resets |

**The loop: Connect → Earn → Boost → Grow**

## Install Instructions

### Android
1. Install from [Google Play Store](https://play.google.com/store/apps/details?id=teneo.node)
2. Log in or create an account

### iOS
1. Join via [TestFlight](https://testflight.apple.com/join/bSUKTYJn)
2. Install **Teneo Protocol Beacon** from TestFlight
3. Log in with your Teneo account

### Windows
1. Download the `.msi` file from the [latest release](https://github.com/TeneoProtocolAI/teneo-node-app-release-beta/releases/latest)
2. Double-click to run
3. If you see "Windows protected your PC":
   - Click **More info**
   - Click **Run anyway**

### macOS
1. Download the `.dmg` file from the [latest release](https://github.com/TeneoProtocolAI/teneo-node-app-release-beta/releases/latest)
2. Open the DMG and drag **Teneo Beacon** to Applications
3. The app is signed and notarized — it should open without issues

### Linux
**Debian/Ubuntu:**
```bash
sudo dpkg -i Teneo\ Beacon_*_amd64.deb
```

**Fedora/RHEL:**
```bash
sudo rpm -i Teneo\ Beacon-*-1.x86_64.rpm
```
